# Bryson Tyler Productions - Backend

Complete backend for Bryson Tyler Productions.

## Features
- JWT Authentication + Roles
- Content Upload with Protection
- KYC via Didit
- NOWPayments (Subscriptions + Coins)
- Coin System + Tipping
- Live Streaming (LiveKit)
- WebSocket Chat
- My Models + Consent System
- Collaboration between Creators
- Password Reset
- Twitter Sharing
- Admin Dashboard

## Setup
1. Copy `.env.example` to `.env`
2. Run `npm install`
3. Run `npm run dev`

## Deployment
- Backend: Render or Railway
- Frontend: Cloudflare Pages
